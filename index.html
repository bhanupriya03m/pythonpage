1) PYTHON CHARACTER SET

Character set is a set of valid characters that a language can recognise.

  • Letters
  • Digits
  • Special Symbols
  • Whitespaces
  • Other Characters

2)TOKENS

The smallest individual unit in a program is known as a TOKENS .

PYTHON has following Tokens :

  • Keywords
  • Identifiers
  • Literals
  • Operators
  • Punctuators
